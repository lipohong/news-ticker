⚠️ CORS Troubleshooting

If you see "CORS Error" messages, RSS feeds are blocked by browser security. Try these solutions:

  • Option 1: Use a CORS browser extension (Chrome: "CORS Unblock" or "Allow CORS")
  • Option 2: Disable CORS temporarily: chrome.exe --disable-web-security --user-data-dir="C:/temp"
  • Option 3: Set up a local proxy server (see README)
  • Option 4: Use RSS feeds that support CORS (BBC, The Guardian work better)
